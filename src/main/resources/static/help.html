搜索指南

本系统提供高效的全文搜索功能，支持两种主要搜索模式：

  • 段落内容搜索 - 搜索关键词出现在段落正文中的内容
  • 事件标题搜索 - 搜索关键词出现在事件标题中的内容

如何使用

  1. 在搜索页面的搜索框中输入您想要查找的关键词
  2. 点击搜索按钮或按回车键开始搜索
  3. 搜索结果将分为段落内容和事件标题两部分显示
  4. 匹配的关键词会以高亮（黄色背景）的形式显示

搜索技巧

关键词选择技巧

  • 使用具体、精确的关键词可以获得更准确的结果
  • 避免使用过于通用的词汇，如“的”、“是”等常见词
  • 对于专有名词，请使用标准写法

结果解读

  • 搜索结果按相关性排序，最相关的内容排在前面
  • 每个结果下方显示相关度百分比，数值越高表示相关性越强
  • 段落结果会显示匹配的文本片段，方便快速预览

常见问题

为什么我的搜索没有结果？

可能是因为您的关键词过于特殊，或者系统中还没有相关数据。请尝试使用更通用的关键词，或者检查关键词的拼写是否正确。当前系统仅包含资治通鉴部分章节内容，尚属内部测试

搜索结果中的高亮部分是什么？

高亮显示的部分是您搜索的关键词在文本中的位置，黄色背景使它们更容易被识别。

如何提高搜索结果的准确性？

使用更具体、更精确的关键词，避免模糊不清的描述。系统会自动对中文进行分词处理，您不需要手动分词。
